# Training: The Closing Slide

The **Closing** slide is the last page of every deck (it's locked to the
end) and the **single home for contact info** — email, phone, and address
live here, not on Next Steps. Its job: ask for the signature and make it
easy to reach us.

---

## 1. Where the content comes from

- **Headline** — per-deck text; clearing it falls back to the closing
  headline saved in **Settings → Company Settings**.
- **Contact info** — email / phone / address sync from Company Settings
  branding. The inspector fields are *overrides*; leave them blank to stay
  in sync with Settings.
- **Validity note** — defaults to *"This proposal is valid for 30 days."*

> **Heads-up:** the AI Edit box can rewrite the headline, CTA wording, and
> taglines — but it can **never** touch the contact info. That's factual
> data and stays synced.

---

## 2. Choose a layout

**In the app:** Inspector → **Layout**. Four options:

| Layout | Looks like | Best for |
|--------|-----------|----------|
| **Blueprint Split** (default) | Left: grayscale blueprint sheet with the glowing HHI logo. Right: bold headline, CTA paragraph, "valid for 30 days," and an orange-topped contact box | The action close — "Securing Your Project Schedule" |
| **Dark Centered** | Warm charcoal, big serif headline, subhead, centered logo, contact lockup, fine print | The classic "Ready to Transform Your Home?" thank-you |
| **Light Logo** | Linen background, large logo, same stack in navy | A softer, brighter goodbye |
| **Photo + Card** | Full-bleed project photo with a centered white card | Ending on the client's own space |

---

## 3. Blueprint Split controls

**In the app (Inspector — these sections appear only on Blueprint Split):**

- **Call to Action → Paragraph** — the close. Default: *"Sign the Design
  Contract to formalize the relationship, initiate the Feasibility Study,
  and secure your place in our production schedule."* Plus a size slider.
- **Contact Box → Bold Line** — default *"Turning Ideas into Ideal
  Spaces."* The box then shows **email | phone** and the address from
  Contact Info. Box Text Size scales the whole box.
- **Blueprint Image** — a built-in floor-plan sheet ships with the app, so
  the layout works with zero setup. **Change** lets you swap in the
  client's *actual* floor plan or CAD sheet from the media library — a
  great personal touch. **Use default sheet** restores the built-in.
- **Logo** — the glowing logo over the blueprint; resize it with the
  shared Logo size slider.

> **Tip:** the headline defaults to "Securing Your Project Schedule" on
> new decks. If you switch an older deck to this layout, retitle the
> headline to match the close.

---

## 4. Dark Centered controls

Matches the reference deck order: big serif **headline** → optional
tagline + subheadline → **logo** → contact lockup (company name, address,
phone | email) → fine print at the bottom.

- **Background Color** — defaults to warm charcoal; any color works.
- **Background Photo** — optional full-bleed photo with a dark overlay.
- **Tagline** — type a single space to hide it (the logo lockup already
  says "Design • Build • Remodel").
- Every text block (headline, tagline, subheadline, contact, validity) has
  its own font / size / style / color controls.

---

## 5. AI Edit

**In the app:** the **✦ AI Edit** box at the top of the inspector. Plain
English — *"make the CTA more urgent,"* *"switch to the dark layout and
warm up the background,"* *"shorten the validity note."* **↶ Undo**
reverts. Contact info is off-limits to the AI by design.

---

## 6. Quick-start checklist

1. Stay on **Blueprint Split** for the action close (or pick the layout
   that matches the deck's energy).
2. Confirm the **contact info** matches Company Settings (blank overrides
   = always in sync).
3. Optional: swap the **Blueprint Image** for the client's real floor plan.
4. Read the **CTA paragraph** out loud — it's the last thing they see.
5. Check the **validity note** date policy still matches how you quote.
