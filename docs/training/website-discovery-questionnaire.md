# Website Discovery Questionnaire — Team Portal

A shared online form where the marketing team answers the **Custom Website
Build: Discovery Questionnaire v2** — 73 questions across 12 sections covering
brand, content, the AI Estimator, the Repair Pricer, integrations, and the new
booking + RENDR scan flows. Everyone works in the same document: answers save
automatically, you can attach files and links to any question, and the whole
thing exports as one Markdown file when it's done.

## Getting in

**In the app:** open the invite link you were sent — it looks like
`https://app.hhi-builders.com/discovery?k=...`. The code after `k=` is the
access code; the link signs you in automatically.

- If you land on the **access code screen**, your link was cut off in
  copy/paste. Ask for the full link or paste the access code into the box.
- No app login or account is needed — this page is separate from the
  proposal-app sign-in.

**Heads-up:** anyone with the link can read and edit the answers. Share it
with the team working on the website, not outside the company.

## First thing: add your name

**In the app:** type your name in the **Your name** box at the top.

Every answer, link, and file you add is tagged with this name, so the team can
see who said what. The browser remembers it — you only enter it once per
device.

## Answering questions

**In the app:** click into any answer box and type. Short answers and bullets
are fine.

- Answers **save automatically** about a second after you stop typing — watch
  for the green **Saved ✓** under the box. There is no Save button.
- Not sure yet? Click **Mark TBD** — “TBD” is a real answer, and it counts
  toward progress so open items don't look untouched.
- Each question shows **who last edited it and when**, so you can pick up
  where a teammate left off.

**Tip:** start with the starred sections — **Section 3 (AI Estimator),
Section 4 (Repair Pricer), and Section 11 (Strategic decisions)**. Those
unlock the highest-value build work first. The ⭐ chips in the header bar jump
straight to them.

## Attaching files and links

Some questions ask for things like the brand guide, logo files, reference
sites, or the current Repair Pricer prompt.

**In the app:** under any answer box —

- **+ Attach file** — pick one or more files (PDFs, images, video, zip —
  up to 500 MB each). They upload to HHI's file storage and appear under the
  question with a 📎.
- **+ Add link** — paste a URL (a label is optional) and click **Add**.
  Links appear under the question with a 🔗 and open in a new tab.

Click the **✕** next to any file or link to remove it.

## Tracking progress

- The header shows **answered / 73** with an orange progress bar.
- Each section chip in the header shows its own count (e.g. `3/8`) and turns
  orange when the section is complete. Click a chip to jump to that section.
- A question counts as answered once it has any text, link, or file.

## Working as a team

- Several people can have the page open at once. When you click back into the
  tab, it pulls in teammates' latest answers automatically.
- If two people edit the *same question* at the same time, the last save wins
  — so split up by section rather than typing over each other.

## Exporting the answers

**In the app:** click **Export answers** (top right).

This downloads one Markdown document with every question, answer, link, and
file (as download links), plus who answered and when — formatted to send
straight back to Claude.ai to generate the website build prompts. You can
export at any time; it always reflects the current state.

**Tip:** export periodically as a backup snapshot while the team fills things
in — it's instant and doesn't change anything.
