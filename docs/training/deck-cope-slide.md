# Training: The COPE Slide (Cost of Project Execution)

The **COPE** slide explains everything HHI handles behind the scenes to run the
project — permits, site protection, supervision, systems coordination, final
compliance — that's built into the price. It reassures the client that the number
covers a fully managed job, not just materials and labor.

---

## 1. Where it lives

**In the app:** open a project → **Deck** tab → the **COPE** slide. There's one
COPE slide per project (it's a single project-overhead summary, not per-room).

---

## 2. Choose a layout

**In the app:** Inspector → **Layout**. Four options:

| Layout | Looks like | Best for |
|--------|-----------|----------|
| **Columns** | Classical architectural columns under a beam, each with a title, accent rule, and a few short lines | The most "design-build" look — great default |
| **Hexagon** | A honeycomb of hexagons, each with an icon + title + short description | Distinctive, modern |
| **Icon Columns** | A clean 3-column grid of icon + title + description/bullets | Simple and legible |
| **Annotated Diagram** | A callout list on the left + a hero image on the right | When you have a strong site/process photo |

---

## 3. Edit the categories

**In the app (Inspector):**

- **Title / Section label / Intro** — the slide heading and framing.
- **Categories** — each has a **title**, a short **description**, and optional
  **bullets**. Edit, reorder, add, or remove.
- **Lock styles together** — keeps every category's text styling consistent.

**Heads-up — fields follow the layout.** The inspector only shows the controls a
layout actually uses, so you're never editing something with no effect:

| Field | Columns | Hexagon | Icon Columns | Annotated |
|-------|:------:|:------:|:-----------:|:--------:|
| Title | ✅ | ✅ | ✅ | ✅ |
| Description | ✅ | ✅ | ✅ | ✅ |
| Bullets | ✅ | — | ✅ | — |
| Icon | — | ✅ | ✅ | — |
| Callout Label | — | — | — | ✅ |

(Columns is a classical-column look that intentionally has no icons; Hexagon and
Annotated use the description rather than bullets.)

---

## 4. Icons

The **Hexagon** and **Icon Columns** layouts show an icon per category. You have
two ways to set it:

**In the app:** open a category → **Icon**. Pick one of the built-in icons
(Permit, Shield, Clipboard, Systems, Check) — the currently selected one is
highlighted — or choose **None**.

For project-specific or branded icons, use **Custom / AI Icon** (uploaded brand
icons) or run ✦ **AI Edit → "generate an icon for each category"**, which pulls
from the self-growing icon library (matched or generated, then reused). When an
AI icon is active, a note appears and you can click any built-in to replace it.

---

## 5. AI Edit

**In the app:** the **✦ AI Edit** box at the top of the inspector. Type a change
in plain English — e.g. *"switch to the columns layout,"* *"tighten each category
to one line,"* or *"generate icons for each category."* It can edit the title,
section label, category text/bullets, switch layout, recolor, and generate icons.
**↶ Undo** reverts the last change, and your category wording is preserved unless
you explicitly ask to change it.

---

## 6. Quick-start checklist

1. Open the **COPE** slide.
2. Pick a **Layout** (try **Columns** or **Hexagon**).
3. Tidy the **categories** (titles, descriptions, bullets).
4. Run **AI Edit → "generate icons for each category"** for bespoke icons.
5. Turn on **Lock styles together** so the categories stay consistent.
